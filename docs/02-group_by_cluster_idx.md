# `group_by_cluster_idx`

```python
group_by_cluster_idx(classifications: NDArray[clust_idx_t]) -> list[list[int]]
```

Group sample indices by cluster label.

This helper function converts a 1D array of cluster assignments into an explicit list of clusters, where each cluster is represented as a list of sample indices belonging to it.

It is designed specifically for use with **SPORE**, which guarantees a particular labeling contract.

---

## Labeling assumptions

This function assumes that cluster labels:

* Are **integers**
* Form a **bounded range $\in [-1, N]$**
* May optionally include the **noise label** (`-1`)
* Are generated by SPORE or another algorithm that enforces these guarantees

Because of these assumptions, labels are used **directly as list indices**, without remapping, hashing, or validation.

This function is not suitable for arbitrary clustering outputs (e.g. sparse labels like `{0, 2, 7}`, string labels, or multiple negative values).

---

## Parameters

### `classifications : NDArray[clust_idx_t]`

A one-dimensional array of length `N`, where `classifications[i]` is the cluster index assigned to sample `i`.

---

## Returns

### `list[list[int]]`

A list of clusters, where each cluster is represented by a list of sample indices.

* Empty clusters are omitted.
* If a noise label is present, it appears the last of the returned clusters.
* Clusters are ordered by increasing cluster index (after empty removal).
* If `classifications` is empty, an empty list is returned.

---

## Example

```python
import numpy as np

classifications = np.array([0, 0, 1, -1, 1])
clusters = group_by_cluster_idx(classifications)

print(clusters)
```

Output:

```text
[[0, 1], [2, 4], [3]]
```

Here:

* Samples `0` and `1` belong to cluster `0`
* Samples `2` and `4` belong to cluster `1`
* Sample `3` belongs to the noise cluster (`-1`)
